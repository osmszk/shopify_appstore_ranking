## 初期インストール
### virtualenvで環境つくる
```
$ virtualenv my_env
$ source my_env/bin/activate
(my_env) $ pip install ...
```

### pip install
```
pip install beautifulsoup4 jupyter urllib3 selenium
pip install webdriver-manager
```
